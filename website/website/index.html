CSC 474 Final Project

Chanelle Mosquera & Nick Sciacqua - CSC 474 - Spring 2018 - Christian Eckhardt

Dragon Animation and Path Editor Tool

For our final project, we implemented a rigged dragon mesh that follows along a path. There's two animations for the dragon, running and flying. Another feature is the path editing tool - points along the path can be added and tested during run time.
